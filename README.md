Cats Gallery Application
=========================

A web application for displaying images of cats

Installation
------------
>npm install

#Dependencies installed are

"express": "4.14.0",
"path": "0.12.7",
"request": "2.72.0"

#Running the Server

In the root directory,
>npm start

Usage
-----

>Visit Cat gallery using http://127.0.0.1:8085/
